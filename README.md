Smart Customer Analysis Tool

AI Project Management (Mock Project)

Project Overview

This repository documents the project management artifacts and discussions for the Smart Customer Analysis Tool, an Artificial Intelligence project developed for academic and management purposes only.

The focus of this project is AI project planning and management, not system implementation or model development.
No coding implementation is required unless explicitly stated.

Project Objectives

Apply project management principles to an AI-based project

Demonstrate all Project Management Process Groups

Practice team collaboration and discussion using GitHub

Produce structured documentation using Microsoft Project, Word, and PowerPoint

Project Scope

In scope

Project Charter

Work Breakdown Structure (WBS)

Project Schedule and Cost Estimation

Risk, Resource, and Monitoring Plans

Management discussion and decision documentation

Final report and presentation materials

Out of scope

Model training or deployment

Application development

Data collection and coding implementation

Project Management Process Groups

This project follows the PMBOK Process Groups:

1. Initiating

Business Case

Project Charter

Stakeholder Identification

2. Planning

Scope Statement

Work Breakdown Structure (WBS)

Schedule and Cost Estimation

Risk and Resource Planning

3. Executing

Coordination of planned activities

Team communication and documentation updates

4. Monitoring & Controlling

Progress tracking against plan

Issue identification and corrective actions

Change documentation (if applicable)

5. Closing

Formal project completion

Final documentation and acceptance

Lessons learned

Tools & Software Used

Microsoft Project – scheduling, WBS, and cost estimation

Microsoft Word – project report documentation

Microsoft PowerPoint – project presentation
